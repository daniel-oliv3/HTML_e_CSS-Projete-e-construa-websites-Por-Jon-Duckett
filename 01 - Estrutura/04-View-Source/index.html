Como visualizar o código-fonte desta página

Se você for para o menu Exibir do seu navegador, verá uma opção que permite ver o código-fonte por trás de uma página da web.

Quando a web começou a decolar, essa era uma forma muito popular de as pessoas aprenderem como as páginas da web foram construídas.

Hoje, as pessoas ainda costumam ver a origem das páginas para aprender como uma página da web foi construída.
